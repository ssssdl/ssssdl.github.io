改用kali

之前就听说各种大佬已经不用windows改用linux，于是我也尝试了一波，大致分为美化和运行Windows应用两部分

关于美化

linux的美化除了常见的设置主题壁纸终端配色，还有修改桌面启动器这一说，感觉大概就像Android手机可以换启动器一样，下面主要介绍两种常见的桌面启动器

0x00 kde桌面

apt-get install kde-plasma-desktop

安装中间会有一个选择默认桌面的过程，直接选择kdm就行了 ，安装完成注销，或者重启，在输入密码界面，选择上kde离子桌面，然后就可以进系统了，在这个桌面下的应用商店把软件更新一下

注意root下是打不开默认的文件管理器的

可能要装一下 dolphin（文件管理器）和deepin-terminal（一个挺不错的终端），apt-get安装就可以
就这样吧，感觉不错的话还可以把gnome删了

接下来自己配置一些主题插件就可以了
附一张效果图，感觉还可以，

运行Windows应用

我知道有两种途径：wine和安装linux版的Windows软件

0x00 wine

apt-get install wine32
也许会用到dpkg --add-architecture i386(安装32位库)
安装wine，这个东西我感觉就相当于mingw，mingw是在Windows上运行linux的一些环境，wine正好相反，他会生成/root/.wine的路径，下模仿Windows分盘，C盘下有windows，Program Files等等一些文件夹
运行exe命令 wine 文件名.exe
用这个安装Windows软件也可以
wine还可以调整Windows兼容版本什么的，但是 因为桌面问题，没有点开这个应用的图形化界面，，，，
